# 🎓 EDU - Career Search & Recommendation Platform

A full-stack web application developed using **Flask**, **PostgreSQL**, **HTML**, **CSS**, and **JavaScript** that helps students discover careers, courses, colleges, and career transition pathways through an interactive and user-friendly interface.

---

## 📌 Project Overview

SPARK is designed to simplify career exploration for students. Users can search professions, explore related careers, view detailed information, discover educational pathways, and receive guidance for career transitions.

---

# ✨ Features

- 🔍 Smart Career Search
- 📚 Profession Details
- 🔗 Related Career Recommendations
- 📈 Top Searched Careers
- 🎯 Career Categories
- 🔄 Career Transition Pathway
- 🤖 Smart Advisor Module
- 📧 Support & Contact System
- 🗄 PostgreSQL Database Integration
- ⚡ Fast Search using Flask APIs

# 🚀 Main Modules

### 🔍 Explorer
Search professions and view detailed career information.

### 🤖 Smart Advisor
Provides guidance to help users explore suitable career options.

### 📚 Categories
Browse careers, courses, colleges, and examinations.

### 🔄 Transition Pathway
Helps users explore pathways to switch from one career to another.

### 📧 Support
Allows users to submit queries and receive confirmation emails.

---

# 💾 Database

The project uses PostgreSQL with relational tables for:

- Profession Information
- Profession Relationships
- Career Links
- User Support Requests

# 🔧 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/EDU.git
```

Install dependencies

```bash

```

Run the application

```bash
python app1.py
```
